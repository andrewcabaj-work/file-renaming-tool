**File Renaming Tool** </br>
Basic application </br>
Adds or removes a given input to the beginning or end of a filename for all files in a specified folder. <br/>
Prefix inserts/removes text to or from the beginning. </br>
Suffix inserts/removes text to or from the end without disturbing the file extension. </br>
To specify the folder, either copy the path or browse via file explorer.
